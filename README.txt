DEP24-P01-AlumniPortal
A way connect alums with each other, to current and prospective students and most importantly to the institute.


1. Unzip the given folder
2. Run command "npm i" in the server folder 
3. Run command "node index.js" in the server folder
4. Run command "npm i" in the my-react-app folder
5. Run command "npm start" in the my-react-app folder
6. The local host has been started and the website will load in your browser
